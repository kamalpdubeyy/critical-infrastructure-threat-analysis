# Disinformation Campaign Analysis

This folder contains analysis of the psychological and disinformation components used during the simulated hybrid attack scenario.

## Areas Analysed
- Fake news propagation
- Social media manipulation
- Deepfake-style media
- Public panic amplification
- Information warfare techniques

## Focus
The analysis examines how misinformation can be used alongside cyberattacks to amplify societal disruption and undermine trust in critical infrastructure.
