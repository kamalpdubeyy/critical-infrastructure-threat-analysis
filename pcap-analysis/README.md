# PCAP Analysis

This folder contains packet capture (PCAP) analysis files and investigation notes related to the simulated attack against a water treatment facility.

## Analysis Areas
- Reconnaissance activity
- SMB lateral movement
- ICS/Modbus communication
- Command & Control (C2) traffic
- Potential sabotage attempts

## Tools Used
- Wireshark
- tshark
- Zeek

## Focus
The analysis focused on identifying attacker behaviour across IT and OT environments within a simulated critical infrastructure network.
